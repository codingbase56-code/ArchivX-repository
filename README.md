# ArchivX repository
ArchivX is a project of Library management.In this project, I had used file handling and Java Swing in a creative way. The project allows you to store books on this GUI app
which provides a user friendly UI for the users to interact. It had got many promising features which I will tell about later on. This project was actually a challenge given
by ChatGpt for testing my coding skills in the field of GUI apps. This program has also got a interesting feature called the pass. The pass feature is about simplicity in a 
code base which primarily stores its data in files. I think it is an epitome of simplicity when it comes to creating a project which stores data in files. It is also very 
simple to operate and use. When a person who is a non-librarian visits the app and click on th book title then, he gets a numeric value called pass, after the generation
of pass, the burrower must note it down somewhere so, that he/she doesn't forget it, after that the librarian can give the book to them. The pass is necessary in the time
of book return, because it acts as a verifier to check whether the authentic user is returning the book or not. The features of this app is :
1. Adding the books
2. Issuing the books
3. Searching the books
4. Returning the books
5. The pass
6. The overall information of books available and issued in the form of a dialog box.

Suggestions and feedbacks are always welcome. I think that this project has elevated my knowledge about Java Swing and file handling. It has also taught me a piece of important
things that are: Crtical thinking, spatial thinking, Creativity and Problem solving.
